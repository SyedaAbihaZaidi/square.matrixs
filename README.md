# Square Matrix Sums (Java)

## Description
This Java program takes the dimension of a square matrix and inputs all elements.  
It calculates and displays:

- Sum of each row
- Sum of each column
- Sum of forward diagonal
- Sum of reverse diagonal

## Concepts Used
- Java 2D Arrays
- Loops
- Scanner for input
